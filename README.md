# Problem Statement

Hospitals often update doctor availability manually.

When a doctor becomes unavailable:

* Existing appointments are affected.
* Patients are not informed immediately.
* Replacement doctors are assigned manually.
* Operational teams spend significant time validating schedules.

The objective of this demo is to show how automation can identify operational issues, estimate impact, and recommend actions before patients are affected.
